Issue 5412: Auto-completion for CSS selectors

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Select issue5412.html from the Location Menu
    ⇒ One CSS rule should be listed
  4. Open the rule for editing, and enter various things to check completions:
  5. "#" ⇒ "#content"
  6. "." ⇒ ".a"
  7. "s" ⇒ "span"
  8. "#e" ⇒ "#element1"
  9. "#element1." ⇒ "#element1.element1"
  10. "#element1#" ⇒ "#element1#"
  11. ".element1." ⇒ ".element1."
  12. "section > h" ⇒ "section > h1"
  13. "section>h" ⇒ "section>h1"
  14. "#element1 .a." ⇒ "#element1 .a.second"
  15. "span, #element1 .a." ⇒ "span, #element1 .a.second"
  16. ".a[" ⇒ ".a["
  17. ".a[t" ⇒ ".a[t"
  18. ".a[data-test=test]." ⇒ ".a[data-test=test].second"
  19. ":" ⇒ ":hover"
  20. "a:hover::a" ⇒ "a:hover::after"

Expected result

  • There should be (context-sensitive) auto-completion for CSS selectors.
