Issue 5358: Issue summary

Only cell

Steps to reproduce

  1. Open Firebug and switch to the HTML panel
  2. Copy this text: <td id="middleCell">Foo!</td>
  3. In the node tree, right-click on the #onlyContent element, and select Paste > Replace Node
  4. Copy this text: <b>Middle</b>
  5. In the node tree, right-click on the #middleContent element, and select Paste > Replace Content
  6. Copy this text: <td>Left</td>
  7. In the node tree, right-click on the #middleRow element, and select Paste > As First Child
  8. Copy this text: <td>Right</td>
  9. In the node tree, right-click on the #middleRow element, and select Paste > As Last Child
  10. Copy this text: <tr><td colspan="3">Top</td></tr>
  11. In the node tree, right-click on the #middleRow element, and select Paste > Before
  12. Copy this text: <tr><td colspan="3">Bottom</td></tr>
  13. In the node tree, right-click on the #middleRow element, and select Paste > After
  14. In the node tree, right-click on the <html> element, ans select Paste: the "Replace Node", "After" and "Before" items should be disabled
  15. Check the result (see below) and go on to issue5358_xml.html

Expected result

You should see this on the top of the page:

Top
Left Middle Right
Bottom
