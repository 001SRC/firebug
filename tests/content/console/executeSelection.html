Issue 5535: Execute Command Selection

This test case is intended for command editor
Florent FAYOLLE, florent.fayolle69__AT__gmail__DOT__com

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Switch to the editor mode
  4. Type the following commands :
    var a = "no selection";
    var b = a || "selection";
    console.log(b);
  5. Select the second and the third line
  6. Click on "Run".
  7. The Console panel should display :
    >>> selection
  8. Collapse the selection
  9. Click on "Run"
  10. The Console panel should display :
    >>> no selection
  11. Switch back to Command Line
  12. Select var b = a || "selection"; console.log(b);
  13. The Console panel should display :
    >>> no selection
